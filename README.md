# 📍 Mapa Przygód

Interaktywny dziennik podróży, który automatycznie porządkuje zdjęcia z wyjazdów na mapie.

### 🌟 Co to potrafi?
* **Automatyczna lokalizacja:** Zdjęcia umieszczone są tam, gdzie zostały zrobione.
* **Filtrowanie wspomnień:** Można szybko wyświetlić zdjęcia tylko z konkretnego regionu (np. "Kraków i okolice", "Tatry", "Bałtyk").
* **Pełna galeria:** Każdy punkt na mapie to miniatura, która po kliknięciu otwiera wspomnienie w pełnym rozmiarze.

### 🛠️ Technologia
Zbudowane z pasją przy użyciu **Leaflet.js** (mapy) oraz **Supabase** (bezpieczna baza w chmurze).
